# 시작하기

**새 문서 생성기**에 오신 것을 환영합니다! 이 가이드는 아름다운 다국어 문서 사이트를 만드는 방법을 안내합니다.

## 개요

새 문서 생성기는 다음 기능을 제공하는 레지스트리 기반 시스템입니다:

- 🚀 **빠름**: 최적의 성능을 위해 Astro로 구축
- 🌍 **다국어**: 15개 이상의 언어를 즉시 지원
- 🔍 **검색 가능**: Pagefind로 구동되는 즉각적인 검색
- 📦 **버전 관리**: 여러 버전을 쉽게 전환
- 🎨 **사용자 정의 가능**: CSS 변수를 사용한 테마 시스템
- ♿ **접근 가능**: WCAG 2.1 AA 준수

## 주요 기능

### 레지스트리 기반 아키텍처

모든 문서 구조는 JSON 레지스트리 파일에 정의됩니다. 이를 통해 다음을 제공합니다:

- 프로젝트 구조의 **단일 진실 공급원**
- JSON 스키마 검증을 통한 **타입 안전** 구성
- **버전 제어**에 친화적인 형식
- 기존 문서에서의 **쉬운 마이그레이션**

### 공유 패키지

생성기는 재사용 가능한 패키지로 구축되었습니다:

| 패키지 | 설명 |
|-------|------|
| `@docs/generator` | 레지스트리 로딩, 라우트 생성, 사이드바/사이트맵 생성 |
| `@docs/ui` | 30개 이상의 Astro UI 컴포넌트 (네비게이션, 사이드바, 카드 등) |
| `@docs/theme` | 디자인 토큰, CSS 변수, 색상 팔레트 |
| `@docs/i18n` | 15개 언어를 위한 국제화 유틸리티 |
| `@docs/versioning` | 버전 선택기 및 차이 비교 컴포넌트 |

### 검색 기능

고급 검색 기능에는 다음이 포함됩니다:

- **패싯 검색**: 프로젝트, 버전, 언어별 필터링
- **페이지네이션**: 페이지당 10개 결과로 탐색
- **하이라이팅**: 결과에서 검색어 강조
- **빠른 인덱싱**: Pagefind 사용

## 빠른 시작

### 1. 설치

```bash
# 의존성 설치
pnpm install @docs/generator @docs/ui @docs/theme @docs/i18n

# 또는 모노레포 워크스페이스 사용
pnpm install
```

### 2. 레지스트리 생성

`registry/docs.json` 파일 생성:

```json
{
  "$schemaVersion": "1.0.0",
  "projects": [
    {
      "id": "my-docs",
      "displayName": {
        "ko": "내 문서"
      },
      "languages": [
        {
          "code": "ko",
          "displayName": "한국어",
          "default": true
        }
      ],
      "versions": [
        {
          "id": "v1",
          "name": "버전 1.0",
          "isLatest": true
        }
      ]
    }
  ]
}
```

### 3. 콘텐츠 생성

`src/content/docs/`에 MDX 파일 생성:

```
src/content/docs/
└── my-docs/
    └── v1/
        └── ko/
            └── guide/
                └── getting-started.mdx
```

### 4. 빌드 및 배포

```bash
# 개발 서버
pnpm dev

# 프로덕션 빌드
pnpm build

# 미리보기
pnpm preview
```
