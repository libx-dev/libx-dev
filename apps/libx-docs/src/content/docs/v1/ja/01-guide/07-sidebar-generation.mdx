# サイドバー自動生成の仕組み

サイドバーは実ファイル構造から自動生成されます（`utils/sidebar-generator.ts`）。手動設定は不要で、次のルールで並びます。

## 並び順の基本

1. ディレクトリ名の番号（例: `01-guide`, `02-reference`）
2. ファイル名の番号（例: `01-getting-started.mdx`, `02-...`）

番号はURLにも含まれるため、命名時に確定させると後工程が楽です。

## frontmatterでの上書き

カテゴリや項目順はfrontmatterで微調整可能です。

```yaml
---
title: "ページ"
category: "guide"        # パス推定を上書き
categoryOrder: 1         # カテゴリ列の順序（小さいほど上）
order: 3                 # カテゴリ内の順序
---
```

<Alert type="note">カテゴリ名の翻訳は `project.config.json` の `translations.*.categories` を参照します。</Alert>
