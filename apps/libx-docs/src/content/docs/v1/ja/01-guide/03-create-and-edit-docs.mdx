# ドキュメントの作成と編集

Libxでは「スクリプト＋命名規則」で迷わずドキュメントを増やせます。

## 作成スクリプトを使う

```bash
# 非対話（カテゴリ名とタイトルを直接指定）
node scripts/create-document.js libx-docs ja v1 guide "セットアップと構成"

# 対話モード（おすすめ）
node scripts/create-document.js libx-docs ja v1 --interactive
```

スクリプトは適切な番号プレフィックスとファイルを生成します。

## 命名規則（重要）

- ディレクトリ: `01-guide`, `02-reference` のように `nn-` を付与
- ファイル: `01-getting-started.mdx` のように `nn-` を付与
- サイドバー順は「ディレクトリ番号 → ファイル番号」で決定

## 直接手で作成する場合の雛形

```mdx
---
title: "ページタイトル"
description: "簡単な説明"
order: 3
---

# 見出し

本文…
```

<Alert type="note">`title` は必須。`order` は同一ディレクトリ内の表示順を微調整したいときに使います。</Alert>
