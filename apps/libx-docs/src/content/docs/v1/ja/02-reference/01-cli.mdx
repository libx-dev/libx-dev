# Frontmatter リファレンス

`apps/libx-docs/src/content/config.ts` のスキーマに基づくfrontmatterキーと用途です。

## 必須

- `title`: ページタイトル

## 推奨

- `description`: 検索・カード用の説明
- `order`: セクション内の並び順（小さいほど上）
- `prev`/`next`: 手動のページネーション

```yaml
---
title: "ページ"
description: "説明"
order: 2
prev: { text: "戻る", link: "/v1/ja/01-guide/01-getting-started" }
next: { text: "次へ", link: "/v1/ja/01-guide/02-setup-and-structure" }
---
```

## カテゴリ関連

- `category`: カテゴリ名（未指定時はパスから推定）
- `categoryOrder`: カテゴリ列の順序（小さいほど上）

## メタ情報

- `pubDate` / `updatedDate`: 日付
- `author`: 著者名
- `tags`: タグ配列
- `draft`: 下書き（trueで非公開運用がしやすい）

## ライセンス

- `licenseSource`: 使用する出典ID
- `customAttribution`: 追加の帰属表示
- `hideAttribution`: 帰属表示を隠す（true）
