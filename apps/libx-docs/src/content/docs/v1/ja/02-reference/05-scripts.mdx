# スクリプト リファレンス

主要スクリプトの実用的な使い方を抜粋します。

## create-document.js

```bash
# 非対話
node scripts/create-document.js libx-docs ja v1 guide "セットアップと構成"

# 対話モード
node scripts/create-document.js libx-docs ja v1 --interactive
```

番号やfrontmatterの雛形を自動生成します。

## create-version.js（参考）

```bash
node scripts/create-version.js libx-docs v2
```

今回のv1ドキュメントでは利用頻度は低めです。

## add-language.js（参考）

```bash
node scripts/add-language.js libx-docs ko --template-lang=ja --auto-template
```

言語ツリーを追加し、必要に応じてテンプレートを複製します。

## sync-content.js（高度）

```bash
# バリデーションのみ
node scripts/sync-content.js --validate-only

# 実行内容の確認
node scripts/sync-content.js --dry-run
```

外部リポジトリ（libx-docsコンテンツ）との同期・検証を行います。
