# project.config.json リファレンス

`apps/libx-docs/src/config/project.config.json` はサイト動作の要です。

## basic

- `baseUrl`: ベースパス（例: `/docs/libx-docs`）
- `supportedLangs`: サポート言語
- `defaultLang`: 既定言語

## translations

- 言語ごとの表示名と説明
- `categories` にカテゴリ名の翻訳（例: `guide`, `reference`）

## versioning

- `versions`: `{ id, name, date, isLatest }[]`

## licensing

- `sources`: 出典ごとのID・ライセンス種別等
- `defaultSource`: 既定の出典ID
- `showAttribution`: フッターの帰属表示のON/OFF

必要最小限は `basic.baseUrl` と `translations.categories` の整備です。
